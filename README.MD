# Scope

Tool for remote analysis of your application

## Usage

Clone the source locally:

```sh
$ git clone https://github.com/kioltk/scope-app
$ cd scope-app
```

Install project dependencies:

```sh
$ npm i
```

Start the app:

```sh
$ npm start
```

For build:

```sh
$ npm run-script build
```
